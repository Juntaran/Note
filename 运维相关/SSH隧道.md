# SSH隧道

*2017.7.20*


> SSH 隧道就是利用 SSH 进行端口转发，从而实现流往某端口的数据被加密后传向另一机器,
> 这个过程形似构造了一条通道，因此也称之为 SSH 隧道(SSH Tunnel)

### SSH 隧道的类型

1. 动态端口转发(socks 代理)
2. 本地端口转发
3. 远程端口转发

### 动态端口转发

- 条件:

> host1 ---ssh---> host2

- 目的：

> 让 host1 某端口的数据发往 host2，host2 根据其应用程序协议发出到指定地址，就好像是从 host2 直接发出的数据

- 操作：

    host1$ ssh -D 127.0.0.1:7070 user@host2
